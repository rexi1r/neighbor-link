Step 4: Access for Neighbors
Dashboard Next
In this step, we first configure the Guest Wi-Fi settings to allow neighbors to access Starlink. Then, we create individual accounts for each neighbor (user) with a personal username and password.

در این مرحله، در ابتدا وایفای Guest را برای دسترسی همسایگان به استارلینک تنظیم می‌کنیم. بعد برای هر کدام از همسایگان (کاربرها) یک حساب شخصی همراه با نام کاربری و پسورد می‌سازیم.
Instruction:
  • Make sure the Guest Wi-Fi is enabled.
  • You can change the guest network name. (Your neighbors will use this SSID to connect to the internet.)
  • Choose a password.
  • Go to User Management and create a unique username and password for each neighbor (user).
  • Provide the SSID, password, and individual usernames and passwords to your neighbors. Once they are connected to your network, they will need to visit the following portal link to authenticate using their username and password: http://192.168.3.1:2027/splash.html
  • You can review the user statistics at the bottom of the page.
  • مطمئن شوید که Guest Wi-Fi فعال باشد.
  • می‌توانید نام شبکه مهمان را تغییر دهید. (کاربرها از این SSID برای اتصال به اینترنت استفاده خواهند کرد.)
  • یک پسورد برای شبکه مهمان انتخاب کنید.
  • به User Management بروید و برای هر همسایه (کاربر) نام کاربری و پسورد مختص خود را بسازید.
  • اطلاعات وای‌فای (SSID و پسورد) و نام کاربری و پسورد هر کاربر را با آن‌ها به اشتراک بگذارید. بعد از اتصال، آن‌ها باید به لینک زیر مراجعه کرده و نام کاربری و پسورد خود را وارد کنند: http://192.168.3.1:2027/splash.html
  • می‌توانید آمار مصرف کاربران را در پایین صفحه مشاهده کنید.
